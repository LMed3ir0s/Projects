# Frontend

Responsável por exibir os dados enviados pelo backend e apresentar o dashboard ao usuário.  
Fluxo simples: a página inicial chama a API -> recebe as métricas -> renderiza os componentes do dashboard.

---

# Componentes

- **Buscar dados processados** (temperatura, umidade, velocidade do vento, etc.)  
- **Popular os componentes do dashboard**  
- **Atualizar métricas e gráficos automaticamente**  

Esses dados alimentam os componentes visuais que exibem:
- Métricas resumidas  
- Gráficos  
- Tabelas  
- Cards informativos
